# Security Policy

## Supported Versions

This is an [individual project (iP)](https://nus-cs2113-ay2526s1.github.io/website/admin/ip-overview.html) for
[NUS' "CS2113: Software Engineering & Object-Oriented Programming" course](
https://nusmods.com/courses/CS2113/software-engineering-object-oriented-programming). As such, the code will only be
actively maintained during the iP. Once the iP is completed, the code will no longer be maintained.

| Version           | Supported          |
|-------------------|--------------------|
| Latest commit     | :white_check_mark: |
| All other commits | :x:                |

## Reporting a Vulnerability

To report a security vulnerability, please [open an issue in the GitHub repository](
https://github.com/yikjin/ip/issues/new). Updates will be given in the issue itself.
